# ValorantDRPC
 
## Usage
- Run the program instead of launching VALORANT
     - If VALORANT is not running, the program will launch it for you
- If VALORANT is already running, launch the program and the presence will start

## Features
- Shows all ranks
- Shows all maps
- Shows all agents
- Shows your RR
